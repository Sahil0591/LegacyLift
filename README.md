# LegacyLift

LegacyLift lives under the `legacylift/` workspace directory.

```text
legacylift/
├── client/      # Next.js workbench
├── server/      # FastAPI API, SQLite/Postgres persistence, GitHub ingestion
├── extension/   # Chromium GitHub Decision Overlay
└── plans/       # Numbered implementation plans and handoffs
```

Start with [`legacylift/README.md`](legacylift/README.md) for local setup, database configuration, GitHub App setup, extension loading, and verification commands.
